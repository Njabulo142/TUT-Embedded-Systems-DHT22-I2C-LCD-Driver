# TUT Embedded Systems: DHT22 Environmental Monitor

Bare-metal C++ firmware for Arduino Uno implementing DHT22 sensor interfacing and I2C LCD1602 driver. Developed for Tshwane University of Technology Embedded Systems module.

## Technical Implementation
- **DHT22 Protocol:** Digital sensor communication on GPIO pin 7
- **I2C Bus Driver:** LiquidCrystal_I2C library for 20x4 LCD at address 0x27
- **Real-Time Display:** 2-second sensor polling with formatted LCD output
- **Embedded C++:** Hardware abstraction using Wire.h and device libraries

## Hardware
Arduino Uno | DHT22 Sensor | 20x4 I2C LCD Module
